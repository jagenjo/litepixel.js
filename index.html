litepixel.js

Litepixel.js is a 2D GPU accelerated game's library based in litegl.js .

Some of the main features are:

  • Do not generate garbage.
  • Optimized for rendering thousand of sprites.
  • Box2D physics (using emscripten version)
  • PostFX

Download

You can download the full source code from GitHub, or use just the library using this link litepixel.js (compressed version)

Usage

Include the script

<script src="js/litepixel.js"></script>
